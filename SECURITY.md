# Security Policy

ObscuraPrimus is a local file-processing tool and should be treated as experimental security software.

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.x | Yes |

## Reporting Issues

Please report security issues privately to the project maintainer. Include:

- Version and operating system.
- Steps to reproduce.
- Whether the issue affects confidentiality, integrity, or availability.
- Any sample files needed to reproduce the issue.

Do not publish proof-of-concept payloads for active vulnerabilities before a fix is available.
